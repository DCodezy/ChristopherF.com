PicaNotes (Android)


The camera on a smartphone has become a significant tool in note-taking in the modern college classroom. Many of these pictures become scattered and hard to find over time. This application is designed to solve this problem by detecting if a picture is taken during a student's class time, and move it into a convenient, uncluttered separate environment. This provides an easier way of finding these references later on in the semester.
